## Git Payload

Migrate source, tests, protocol and architecture docs, Docker definition, plugin packaging metadata, and portable documentation.

Do not migrate through Git:

- Operator credentials, tokens, recovery keys, or Windows credential-store entries
- Active VibeBus task, inbox, ACK, reservation, delivery, subscription, or runtime SQLite state
- Local `.vibebus-local`, backup directories, databases, build output, or Codex session/plugin cache

The tracked path `tests/credential_vault.rs` is a credential-vault test file by name. It must not be treated as a live secret without content review, and no secret value should be printed during review.

- Windows and Mac runtime databases/credentials remain intentionally separate; cross-device synchronization and vault recovery are not implemented.
